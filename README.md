# SlideMenu

SlideMenu using UITabBarController and Storyboard and Constraint base structure.
- Easy to use
- Easy to implement
- 2 animation style
- Tap gesture recognizer
- Pan gesture recognizer
- Enable/Disable Pan gesture recognizer
- Portrait mode supported
